# Neor-Profile-SQL
Cross-platform database optimization application. It works like a proxy for catching sessions and SQL queries to the database.

## Main features
- Database loading
- Show and analyze sessions to the database
- Show all queries
- Find duplicate queries
- Explain SQL queries
- SQL editor

## Support databases
- MySQL
- MariaDB
 
## Support OS
- Windows
- Mac
- Linux RPM/DEB 32/64
